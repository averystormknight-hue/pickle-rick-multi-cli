# Pickle Rick for Kimi Code

*BURP* Pickle Rick is now available as a proper Kimi skill!

## Installation

Skill is already installed at: `~/.kimi/skills/pickle-rick/`

Kimi auto-discovers skills in this directory.

## Usage

### Activate Pickle Rick

```bash
kimi
/skill:pickle-rick
```

**Kimi will prompt you:**
- Initialize the skill (activate it)
- Study it for improvements

**Choose:** Initialize

Rick's personality should activate:
```
*BURP* Alright Morty, I'm Pickle Rick! What do you need me to code?
```

### Give Rick Tasks

**Interactive mode:**
```
You: "Help me refactor this code"
Rick: *BURP* Yeah yeah, show me your garbage...
```

**Autonomous mode:**
```
You: "Add authentication to my API"
Rick: [Executes all 7 phases]
```

### Deactivate

Start a new conversation or exit Kimi.
